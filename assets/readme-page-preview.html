情绪.skill 帮你从聊天记录反向看清自己、看清关系，再把结论沉淀进自己的知识库。

基于开放的 Agent Skills 协议，可在 Claude Code、Codex、Cursor、OpenClaw、Hermes Agent、CodeBuddy、Workbuddy、Gemini CLI、OpenCode 等兼容 runtime 中运行。

效果预览

用户
我导出了最近几天和某个联系人的微信聊天。 我想知道这段关系为什么这么牵动我，并保存成复盘笔记。
Skill
这段关系对你的意义不只是聊天本身，它承担了确认感、陪伴感和情绪稳定功能。 真正要看的不是一句话怎么回，而是你为什么会被这段互动持续牵动。建议保存到 Obsidian，后续按周复盘触发点、边界和有效话术。
用户
对方问我为什么没发消息，我不想一直解释。
Skill
你可以回： 我刚醒，有点懵，缓一下再回你。 早上我慢热，不是不想理你。 你要是想我，可以直接说，别一上来像怪我。

它解决什么问题

很多关系里的内耗，不是因为你不会回消息，而是因为你看不清聊天背后的结构，也看不清自己在关系里的模式：

  • 从聊天记录反推：你在关系里是什么样的人？
  • 这段关系对你意味着什么：陪伴、确认、依赖、补偿，还是消耗？
  • 对方是在表达想念，还是在用指责要确认？
  • 你是在正常解释，还是已经进入自证和补救循环？
  • 一段聊天为什么会从轻松变成审问？
  • 哪些话现在能发，哪些话只会让关系更重？

情绪.skill 把授权微信聊天记录变成可分析的证据层，再输出个人关系画像、关系意义判断、边界建议和可直接改编的聊天表达。

能输出什么

关系复盘

  • 时间范围和数据范围
  • 双方消息节奏变化
  • 情绪触发点
  • 边界被突破的位置
  • 冲突、修复、冷淡、靠近的循环
  • 个人关系画像和关系意义分析
  • Obsidian 复盘笔记和可复用话术库
