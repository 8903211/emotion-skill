# Phrase Frameworks

Use this reference when producing chat advice or reusable wording.

## Style Rules

- Write like a real WeChat message, not a course script.
- Keep most replies under 1-3 short messages.
- Avoid jargon such as "relationship system", "exam", "strategy", "withdrawal", "low battery" unless the user speaks that way.
- Prefer plain Chinese when the user communicates in Chinese.
- Give options the user can edit, not rigid scripts.

## Pressure Levels

### Low Pressure

Use when the other person is tired, cold, upset, or the relationship is unstable.

Patterns:

```text
我刚醒，有点懵，缓一下再回你。
早上我慢热，不是不想理你。
我会回你，但不想一醒来就开始解释。
今天先轻一点聊，别把气氛弄重。
```

### Light Atmosphere

Use when the other person is receptive and playful.

Patterns:

```text
你这个头像像是会嘴硬的人。
我乱猜一下，你是不是慢热型？
你这句话我可以理解成在乎我吗？
你这个点还醒着，适合被夜宵拯救。
```

### Boundary

Use when the user is over-explaining or being pulled into pressure.

Patterns:

```text
你问我可以，但别带着怪我的感觉，我会紧张。
我不是不理你，我是真的需要缓一下。
我现在继续解释会把话聊重，晚点再说。
这个话题我先停一下，不想越聊越拧。
```

### Repair

Use when the user did create hurt or confusion.

Patterns:

```text
刚才我那句说得有点急，我重新说。
我理解你为什么不舒服，这点我接到了。
我不是要推开你，我是想把话说清楚一点。
这次我会注意，但我也需要我们别用指责开头。
```

## Topic Transformations

### Names and Avatars

Avoid:

```text
你叫什么？
你头像是谁？
```

Use:

```text
这个名字看起来有点冷，但你说话不像冷的人。
你这个头像像是会嘴硬的人。
我先乱猜，你应该有点慢热。
```

### Work and Schedule

Avoid:

```text
几点下班？
忙什么？
```

Use:

```text
搞得你这么晚还没下班，今天有点惨。
你这个点还在忙，适合被夜宵拯救。
看起来今天不是你在上班，是班在上你。
```

### Refusal or Coldness

Avoid:

```text
为什么不行？
你是不是不想理我？
```

Use:

```text
行，那这个先不聊。
收到，我先不追问。
你今天状态不高，我先不闹你。
这次算我提了，下次换你挑。
```

### Emotion

Avoid:

```text
你别生气。
这有什么好烦的？
```

Use:

```text
看把你烦的。
这事确实容易让人冒火。
你现在不是脾气差，是被烦到了。
先把你从这件事里捞出来。
```

### Private or Sexual Topics

Rules:

- Ask permission first.
- Let the other person skip.
- Do not force sensual/sexual frames.

Patterns:

```text
这个有点私人，能问吗？
不想答可以跳过。
这个话题有点往前了，我先刹一下。
你要是这样说，我容易想多。
```

## Risky Phrases To Flag

Mark these as "do not copy directly" unless the relationship clearly supports them:

- forced girlfriend/boyfriend labels
- infantilizing names
- physical punishment jokes
- "trap you / don't let you leave" control jokes
- sexual comments after the other person has not opened that door
- guilt-inducing lines like "I got rejected again"

## Recommendation Format

When giving phrase advice, group:

1. Recommended reply
2. Softer version
3. Playful version
4. Do not send
5. Why this works
